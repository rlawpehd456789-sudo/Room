# マイルーム (My Room)

일본 시장 타겟 인테리어 공유 SNS 플랫폼

## 기술 스택

- Next.js 14
- TypeScript
- Tailwind CSS
- Zustand (상태 관리)
- React Hook Form

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 주요 기능

- 회원가입/로그인
- 온보딩 (관심 스타일 선택)
- 게시물 작성 (사진 업로드)
- 피드 보기 (Masonry 레이아웃)
- 좋아요/댓글
- 프로필 페이지
